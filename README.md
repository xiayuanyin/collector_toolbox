# PLC通讯工具箱

## 介绍

各类PLC的数据采集，数据转换，数据写入，数据展示，数据图表、导出等功能的集合。

目前支持协议：Modbus(TCP/UDP)

后续待支持协议：
- Modbus RTU
- S7comm
- OPC UA
- MC
